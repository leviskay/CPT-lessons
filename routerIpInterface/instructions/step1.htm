Добро пожаловать на урок!

В этом уроке Вы отработаете навыки настройки IP адресов на интерфейсах маршрутизатора Cisco.
Соберите схему согласно рисунка.
Для ПК установите следующие настройки

ПК IP адрес Основной шлюз
ПК1 [[PC1ip]] [[Fa00ip]]
ПК2 [[PC2ip]] [[Fa01ip]]

Затем настройте интерфейсы на маршрутизаторе

Обращаю Ваше внимание!

Для настройки маршрутизатора используйте консольный интерфейс и производите настройку через ПК3
Для этого необходимо соединить ПК3(разъем RS232) и маршрутизатор (разъем Console) кабелем типа Console
Затем произвести настройку через программу Terminal на вкладке Desktop в ПК3.
Настройка маршрутизатора через вкладку CLI в данном уроке заблокирована!

Интерфейс IP адрес
FastEthernet 0/0 [[Fa00ip]]
FastEthernet 0/1 [[Fa01ip]]
