Изучение основ VLAN

Перед началом ВНИМАТЕЛЬНО изучите видео урок по VLAN.
Ниже представлен список основных команд, которые будут использованы в данном уроке
Команда Назначение
(config)#vlan XXX Создать VLAN с номером XXX и перейти в режим его настройки
(config-vlan)#name YYYYY Для выбранного VLAN задать имя YYYYY
(config-if)#switchport mode access Для заданного интерфейса задать режим работы порта (access или trunk)
(config-if)# switchport access vlan vlan-id Для заданного интерфейса задать номер VLAN
